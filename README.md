# vectormap-maxtilt-issue

This application is made to reproduce a render issue with google maps JS Vector Map when on max tilt.


## Project Setup

```sh
npm install
```

## Add API Key
```
add apiKey in mapConfig.ts -> apiKey
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```
